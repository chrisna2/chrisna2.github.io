## 그룹관리

### 그룹 삭제 - groupdel

#### [1] groupdel : 그룹 삭제
- 주 그룹에는 사용지가 없어야 한다. 사용자 먼저 삭제 후 그룹을 삭제
- 파일 중 그룹이 설정한 파일이 없어야 한다.
- 자동으로 그룹 설정 되어있는 파일을 체크 하지 않는다. 사용자가 직접 해당 파일을 다른 그룹으로 설정해야 한다

```s
$ groupdel [option] groupname
```

#### [2] 예제
```bash
### colors 그룹삭제
$ sudo groupdel colors
```
